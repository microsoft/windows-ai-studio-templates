# Windows AI Studio- Templates

## Overview
This repo contains the templates for the [Windows AI Studio](https://aka.ms/AIforWindows) that simplifies generative AI app development by bringing together cutting-edge AI development tools and models from Azure AI Studio Catalog and other catalogs like Hugging Face. You will be able browse the AI models catalog powered by Azure ML and Hugging Face for public models that you can download locally, fine-tune, test and use them in your Windows application.
As all of the computation happens locally, please make sure your device can handle the load.
You can find more information about Windows AI Studio [here](https://github.com/microsoft/windows-ai-studio).
